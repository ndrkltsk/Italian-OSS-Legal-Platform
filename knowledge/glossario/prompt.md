# Prompt

L'**istruzione testuale** data a un LLM per ottenere una risposta. Nel [flusso RAG](/architettura/flusso-rag.md) il prompt combina la domanda dell'utente, le regole di comportamento (es. citare sempre le fonti) e il contesto dei [chunk](/modello-dati/chunk.md) recuperati.

Un prompt riutilizzabile e parametrizzato può essere salvato come [workflow](/funzionalita/workflow.md).
